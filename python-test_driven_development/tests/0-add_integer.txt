# Tests pour la fonction add_integer

## Test 1 : Addition de deux entiers positifs
Input: add_integer(2, 3)
Expected Output: 5

## Test 2 : Addition d'un entier positif et d'un entier négatif
Input: add_integer(-2, 3)
Expected Output: 1

## Test 3 : Addition de deux floats (doivent être convertis en entiers)
Input: add_integer(2.7, 3.4)
Expected Output: 5

## Test 4 : Mélange d'entiers et de floats
Input: add_integer(2, 3.5)
Expected Output: 5

## Test 5 : Utilisation de la valeur par défaut pour `b`
Input: add_integer(10)
Expected Output: 108

## Test 6 : Entrées invalides (chaîne de caractères pour `a`)
Input: add_integer("10", 5)
Expected Output: TypeError: a must be an integer

## Test 7 : Entrées invalides (liste pour `b`)
Input: add_integer(5, [1, 2])
Expected Output: TypeError: b must be an integer

## Test 8 : Aucun argument fourni
Input: add_integer()
Expected Output: TypeError

## Test 9 : Trop d'arguments fournis
Input: add_integer(1, 2, 3)
Expected Output: TypeError
